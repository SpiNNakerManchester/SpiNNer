SpiNNer: SpiNNaker wiring tool
==============================

SpiNNer is a collection of tools for assisting with all stages of the process of
wiring up large SpiNNaker machines, from design to installation verification.

For installation and setup, see the README_, for everything else, continue
reading!

.. _README: https://github.com/SpiNNakerManchester/SpiNNer#installation
